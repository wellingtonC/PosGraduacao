Código desenvolvido em aplicação C# .Net Framework

Os padrões de código adotados foram:
- Variáveis em inglês;
- Primeiro nome das variáveis com a primeira letra minúscula e nome subsequente com letra maiúscula caso o mesmo tenha um segundo nome;
- Try catch para evitar erros na execução do codigo;
- Mensagens para a inserção de dados e tambem retornos para o usuário todos em português.
